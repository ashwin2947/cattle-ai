About

What is this system?

An end-to-end AI pipeline for cattle analysis — from image input to breed identification and physical trait estimation.

This project combines YOLO object detection with EfficientNet classification to identify cattle breeds from images or live video, enhanced with GradCAM heatmaps that explain the model's decision-making process.

  • Automatic cattle detection and cropping from images
  • Deep learning breed classification with confidence scores
  • Visual explanations using GradCAM heatmaps
  • Real-time webcam-based detection pipeline
  • Physical trait estimation (weight & height)

Designed to assist:

🌾 Farmers 🔬 Veterinarians 📊 Researchers 🥛 Dairy Industry

Capabilities

Core Features

A complete AI pipeline covering detection, classification, explainability, and estimation.

🧠

Breed Classification

Identifies cattle breeds from images using a fine-tuned EfficientNet-B4 deep learning model trained on a curated cattle dataset.

🔥

GradCAM Explainability

Generates gradient-weighted class activation maps to visually explain which regions of the image influenced the model's prediction.

📷

Live Camera Detection

Streams webcam feed through the YOLO detection pipeline for real-time cattle identification and breed classification.

⬆️

Image Upload Prediction

Upload any cattle image to receive an instant prediction with breed label, confidence score, and GradCAM overlay.

⚖️

Weight Estimation

Computer vision-based weight estimation using body morphometrics extracted from the cattle image.
